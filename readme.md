# springboot

#### 项目介绍
使用spring boot集成了shiro + redis的前后端分离系统框架demo

#### 软件架构
● 核心框架： spring boot + spring

● mvc框架： spring mvc

● 持久层框架： mybatis

● 数据库连接池： alibaba druid

● 安全框架： apache shiro

● 缓存框架： redis

● 日志框架： logback


#### 使用说明

导入项目后，使用maven下载相关jar包，直接运行SpringbootApplication.java即可
